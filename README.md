# 👾 Retro Pixel Art Maker

An interactive, responsive digital drawing board designed to recreate the classic look of retro arcade games. Users can select colors from a vibrant palette and draw directly onto a dynamic matrix canvas using seamless mouse drag gestures.

## 🛠️ Key Features
- **Fluid Paint Engine:** Supports both single-click painting and fluid click-and-drag mouse tracking to draw continuously like a brush.
- **Dynamic Grid Sizing:** Adjustable canvas resolution allowing users to switch between 16x16, 24x24, and 32x32 dimensions instantly via JavaScript grid algorithms.
- **Precision Eraser Mode:** Built-in toggle selector that dynamically switches colors to transparent for flawless drawing corrections.
- **Arcade Aesthetic UI:** Designed with custom Tailwind gradients, subtle background neon glows, and monospaced font layouts.

## 💻 Tech Stack
- **HTML5:** Modular board structures and standard select dropdown configurations.
- **Tailwind CSS:** Advanced responsive grids, custom hover transitions, and arcade dark-theme typography.
- **Vanilla JavaScript:** Real-time matrix node generation, active drawing flags (`mousedown/mouseenter/mouseup`), and element coloring arrays.
